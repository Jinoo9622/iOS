## UIKit

- 사용자의 인터페이스를 관리하고 이벤트를 처리
  - 제스처 처리, 애니메이션, 그림 그리기, 이미지 처리, 텍스트 처리
  - 테이블 뷰, 슬라이더, 버튼 텍스트필드, 알럿창 등 어플리케이션 화면 구성 요소 

- 기본적으로 MVC 디자인 패턴을 사용
